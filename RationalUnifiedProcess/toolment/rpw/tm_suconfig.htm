Tool Mentor:  Setting Up and Configuring the Rational Process Workbench Tool

Purpose

This tool mentor describes how to set up and configure the Rational Process Workbench (RPW) for your environment.

This tool mentor relates to the Rational Unified Process information:

  • Activity: Set Up Tools
  • Activity: Develop Project-Specific Templates

Overview

The RPW is implemented as an add-on to Rational Rose and requires that Rose is installed and operational on your system before the RPW can be installed. All RPW functions become available when the add-on is enabled in the Rose add-in menu.

RPW helps you develop process models and process component models that are maintained as separate files in the file system, and Process Content Libraries (PCLs) that are a number of files in a directory.

Tool Steps

Perform these steps to set up and configure this tool:

  1. Enable and disable the RPW add-in to Rational Rose
  2. Customize templates

1. Enable and disable the RPW add-in to Rational Rose

RPW is implemented as an add-in to Rose and installs itself as that during the  installation. As a Rose add-in, RPW can be enabled or disabled using Roses add-in manager.

Refer to the topic titled Enabling the Process Workbench add-on in Rational Rose in the RPW online Help for detailed information.

2. Customize templates

Each PCL contains a template section that provides the HTML templates for content files. These template files are themselves standard HTML files, which can be modified using your favorite HTML editor to include or exclude sections, your logo, and so forth.

Once the template is modified, files subsequently created and based on that template will include those modifications.

Refer to the topic titled Customizing templates in the RPW online Help for detailed information.

For More Information

For detailed information on this procedure, see Chapter 5, "Setting up and configuring the RPW tool" in the Developing Process Using Rational Process Workbench manual.
