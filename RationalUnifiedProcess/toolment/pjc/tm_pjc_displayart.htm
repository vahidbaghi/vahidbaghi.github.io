Tool Mentor:  Displaying Artifacts Related to Specific Objects on a Diagram Using Rational ProjectConsole

Purpose

This tool mentor describes how to use Rational ProjectConsole to navigate to particular artifacts from a diagram displayed by Rational ProjectConsole.

Overview

ProjectConsole can display graphics as well as text. In cases where an artifact diagram is displayed, the diagram can be used to instantly navigate to a Web page created for an item displayed in the diagram. For example, a class diagram may show a number of classes in graphics form. This tool mentor demonstrates that by clicking on any class displayed on the graphic, the artifacts page is displayed.

ProjectConsole capitalizes on Java applet technologies to provide you with familiar and multiple forms of navigation.  If you click directly on the text of a tree control node the artifact page associated with that node is displayed. If you expand a tree control node by clicking on the nodes plus-sign, then hyperlinks that would be displayed by clicking on the text of the node are displayed as child nodes, from which you can continue navigating.

This tool mentor is applicable when using Microsoft Internet Explorer (release 5.5 or greater) or Netscape (release 4.6 or later).

Tool Steps

Click directly on the text of a tree control node to perform these steps.

  1. Launch a browser.
  2. Enter the URL for your installation's ProjectConsole site. The URL can be obtained by contacting your ProjectConsole administrator. The ProjectConsole logon screen is displayed.
  3. Enter a valid username and password on this logon screen. The ProjectConsole artifact browser is displayed.
  4. Expand the Developer node and then click on the text on the Design - Current View node.
  5. Click on the Use-Case View hyperlink located in the right frame of ProjectConsole. At this point, the Use-Case View artifact template is displayed and is fully populated with Use-Case View information.
  6. Click on the graphic for the Use-Case Model to display its artifact page. Then click on any of the elements in the Use-Case Model to navigate to additional model information.

Note: The artifacts displayed by ProjectConsole are those defined to be displayed using the Rational ProjectConsole Template Builder. Use ProjectConsole Template Builder to modify the appropriate template to change the displayed artifacts.
