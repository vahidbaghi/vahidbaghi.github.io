Toolkit:  Adding External Links to the Rational Unified Process

Purpose

This toolkit page describes how you add an external link to a page in the Rational Unified Process (RUP). This is useful when you develop Web pages outside of the RUP.

Overview

  1. Find the page's file path
  2. Enter the hyperlink
  3. Use the hyperlink

1. Find the page's file path

Find the file path to the page you want to link to.

  1. Open the RUP and display the page in question in the main frame.
  2. Using Microsoft Internet Explorer: Position the cursor over the page, and right-click with the mouse. Select Properties. Copy-paste the file path.

    Using Netscape Navigator: Position the cursor over the frame, and right-click with the mouse. Select View Frame Info. Copy-paste the file path.

Tips: Finding file paths made easy

The following hyperlink, "display file path", does not point to any page; it is a small utility that will help you find out the file path to any page in the RUP. Test it to find out the file path to this page, by clicking on it now. To be able to use it on any page in the RUP, add it to the 'favorites' using Internet Explorer by using one of the methods mentioned below. Note: This functionality does not work in Netscape.

  1. Place the cursor above the hyperlink and press 'Ctrl'. Drag and drop the hyperlink into the 'favorites' ('bookmarks') by using the right mouse button.
  2. Place the cursor above the hyperlink and right-click. Select 'Copy shortcut'. Now you can paste this into the appropriate place in the Favorites.

Using the utility:

  1. Start the RUP.
  2. Browse to the page you are interested in.
  3. Select the shortcut among the favorites. The file path is now displayed in a small window. Copy-paste the file path from this window.

2. Enter the hyperlink

Enter the hyperlink in a page. Notice the following:

  • Use relative file paths, if possible.
  • Decide whether the hyperlink should open the page in a new browser window or not.
  • Tips: Set the hyperlink's target frame property to 'ory_doc'. Clicking on the hyperlink will then display the page in the main frame of the RUP. (This assumes that the RUP is being displayed in a separate browser window.)

3. Use the hyperlink

Clicking on a hyperlink to a page in the RUP will display the page in a Web-browser window. If the user wants to display the RUP with frames, click on the button labeled "Rational Unified Process" in the right border.
