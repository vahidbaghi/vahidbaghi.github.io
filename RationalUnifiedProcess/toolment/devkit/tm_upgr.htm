Toolkit:  Upgrading the Rational Unified Process

Purpose

This toolkit page describes how to upgrade your development case (and other project-specific HTML-files) to reference a new version of the Rational Unified Process (RUP).

Related RUP activity: Develop Development Case

Overview

To upgrade hyperlinks that reference pages in the RUP:

  1. Handle hyperlinks to new and moved files
  2. Handle hyperlinks to files with unchanged file paths

1. Handle hyperlinks to new and moved files

If files in the RUP have changed names, been moved to different folders, or been removed, please read the Release Notes for information on how to upgrade the hyperlinks referencing those pages.

2. Handle hyperlinks to files with unchanged file paths

If you are upgrading from RationalUnifiedProcess 5.1, 5.5 or 2000, the following steps will make all hyperlinks in your project-specific HTML-file point to the new version of the RUP. There are several ways to do this, however, this procedure uses a simple text editor, such as Microsoft WordPad®:

  1. Locate where the old version, and the new version of the RUP are installed. Let's assume you are upgrading RUP 5.5 to the current version:
    C:/Program Files/Rational/RationalUnifiedProcess5.5
    C:/Program Files/Rational/RationalUnifiedProcess
  2. Locate the file path of the file to upgrade. For example, a development case with the following file path:
    C:/Project/Devcase/devcase.htm
  3. Open the file 'devcase.htm', with a text editor such as WordPad, or edit the file in 'HTML-mode' using an HTML-editor such as Microsoft FrontPage®.
  4. Perform a 'search-and-replace', and replace all occurrences:
    Find: "/RationalUnifiedProcess5.5/"
    Replace: "/RationalUnifiedProcess/"
