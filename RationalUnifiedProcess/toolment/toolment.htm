Rational Unified Process: Tool Mentors

Many activities in the Rational Unified Process (RUP) are supported by software-engineering tools. 

Tool Mentors provide detailed descriptions of how to perform specific process activities or steps, or produce a particular artifact or report, using one or more tools.

Some Rational tools appear in different versions of Rational Suite products. For a list of these particular details, visit our Rational Product Home Page, which requires Internet access.

Tool mentors are provided for the following tools:

  • Rational Administrator
  • Rational ClearCase
  • Rational ClearQuest
  • Rational Process Workbench
  • Rational ProjectConsole
  • Rational PureCoverage
  • Rational Purify
  • Rational QualityArchitect
  • Rational Quantify
  • Rational RequisitePro
  • Rational Robot
  • Rational Rose Tool Set
    • Rational Rose (classic tool mentors)
    • Rational Rose (business modeling toolmentors)
    • Rational Rose RealTime
  • Rational SoDA
  • Rational Suite AnalystStudio
  • Rational Test RealTime
  • Rational TestFactory
  • Rational TestManager
