Report Overview

Models and model elements, have reports associated with them. A report extracts information about models and model elements from a tool. A report presents an artifact or a set of artifacts.

The Rational Unified Process provides a number of sample reports. These can be used as-is or you can use them as a starting point to create your own specific set of reports.
