Use-Case Realization Report		 This report contains information regarding the realization of a use case (<use-case name>) within the design model.

Role:

 Designer

Tool Mentor:

 Create a Use-Case Realization Report using Rational SoDA

Purpose

This report is used by various people interested in the use-case realization, such as designers, testers, and managers.

Brief Outline

  1. Brief Description

    A Brief Description of the realized use case.

  2. Flow of Events Design

    A Flow of Events Design of the use-case realization.

  3. Interaction Diagrams

    Interaction Diagrams of the use-case realization.

  4. Participating Objects

    Objects participating in interaction diagrams of the use-case realization.

    For each object, its class and brief description.

  5. Class Diagrams

    Class Diagrams of the use-case realization.

  6. Derived Requirements

    Derived Requirements of the use-case realization.
