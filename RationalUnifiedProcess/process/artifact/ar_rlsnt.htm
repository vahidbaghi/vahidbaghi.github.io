Purpose

The purpose of the Release Notes is to describe the release.

Timing

The Release Notes are created or updated in the Transition phase, before the product is released.

Responsibility

The Deployment Manager is responsible for writing the Release Notes.

Tailoring

Release notes are mandatory, even for purely internal releases.
