Toolkit: RationalProjectWebExample Directory Structure

The 'RationalProjectWebExample' has the following top level directory structure:

RationalProjectWebExample The frames page "index.htm" is found here.
applet Contains files for the Treebrowser, including project_tree.htm and tree.zip (contains the tree.dat).
artifacts A folder supplied for keeping artifacts.  This is further divided into subfolders for each discipline, and a templates subfolder.
css This is where the CSS file is.
forum Contains files for managing a project discussion forum.
images Contains image files.
library Contains a project library of additional guidelines and reference material. 
main Here most of the project Webs infrastructure files are located.
