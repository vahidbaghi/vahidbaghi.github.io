Toolkit: Introduction

The Process Engineer Toolkit provides supporting tools for configuring a process, including links to example process and project webs.

The Project Web Template

The Rational Unified Process (RUP) is installed along with a template for a project Web site. Click here to view this template.

This toolkit explains how to take the Project Web Template and use it to configure a process for a project.

Detailed Organization-Wide and Project Web Example

A more detailed example is The Wylie College Organization Web Example. This example provides two webs.  One is a organization-wide process web site built as a "shell" on top of the RUP. This web contains hyperlinks to the RUP, as described in  Concepts: Process Configuration. The other web is an example project web which follows the specified process, with hyperlinks to both the organization-wide process and the RUP.

Additional examples of artifacts and project webs may be found on the RUP Resource Center.
