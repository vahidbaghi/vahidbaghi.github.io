Rational Unified Process: Process Engineer Toolkit

The Process Engineer Toolkit is a set of tools and guidelines that complement the Environment Discipline.  The Environment discipline describes the activities of configuring a process, whereas the toolkit provides supporting tools for configuring a process, including examples of process and project webs.

The Process Engineer Toolkit is included as part of the standard Rational Unified Process product.  The toolkit does not include the Rational Process Workbench, which is a separate product for large-scale process configuration.  For more information on the Rational Process Workbench, refer to Rational Process Workbench Tool Mentors.

The toolkit is organized as follows:

  • Introduction
  • User's Guide
  • Directory Structure
  • System Requirements
